==== 使用事件

这里看看事件的使用。我们将在<<cdb_screen_controller>>中创建的界面中添加对图形项目点击事件的处理。在 IDE 中打开界面控制器并且注入图表。为了显示通知，在界面控制器中注入了 `Notifications` bean。

[source, java]
----
@Inject
private Notifications notifications;

@Inject
private SerialChart serialChart;
----
